# POSH.

A sample landing page of a Real State Company built using React.

## Installation

### Clone the repo.
```bash
git clone https://github.com/raffyamoguis/posh.git
cd posh
```
### Install dependencies.
```bash
npm install
```

### Start server.
```bash
npm start
```
